Problem Statement:



Total Expenses for the Event:



An Event Management System's primary function is budgeting. One of the key tasks of the system is to estimate the total expenses incurred by an event and calculate the percentage share of each expense category (branding, travel, food, and logistics) in relation to the total expenses.



Nikhil, the founder of "Pine Tree," wanted to include this budgeting functionality in his company's Proton Event Management System. He has requested your help in writing a program that will calculate the total expenses for an event and the percentage distribution of each of the expenses involved in planning and executing the event.



The program should get the branding expenses, travel expenses, food expenses, and logistics expenses as input from the user and calculate the total expenses for an event and the percentage rate of each of these expenses.

Input format :
The first line of input will contain a floating-point number representing the branding expenses for the event.

The second line will contain a floating-point number representing the travel expenses for the event.

The third line will contain a floating-point number representing the food expenses for the event.

The last line will contain a floating-point number representing the logistics expenses for the event.

Output format :
The output should display the total expenses for the event and the percentage share of each of the expense categories. The total expenses should be printed with two decimal places. Similarly, the percentage share for each category should be printed with two decimal places.



The first output line should display the total expenses for the event in the following format:



"Total expenses: Rs.<total_expenses>"



The next four output lines should display the percentage of each of the expenses in the following format:



"Branding expenses percentage: <percentage_branding>"

"Travel expenses percentage: <percentage_travel>"

"Food expenses percentage: <percentage_food>"

"Logistics expenses percentage: <percentage_logistics>"



Refer to the sample output for formatting specifications.
